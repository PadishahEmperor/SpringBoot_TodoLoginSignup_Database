# SpringBoot_TodoLoginSignup_Database
Maven Project, Import/Open with Ide and Run.
Created a Todo WebApp with Database capabilty with Spring/SpringBoot JDBC.
MVC design with Spring Boot and ThymeLeaf.
Used packages for different Classes categories like Model, Services, Repositories etc.
